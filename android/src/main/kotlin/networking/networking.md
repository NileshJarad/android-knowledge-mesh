### HTTP status code

#### Success code

| HTTP code | Meaning         | 
|-----------|-----------------|
| 200       | Server success  | 
| 201       | Created         |
| 204       | No content      |
| 206       | Partial content |


#### Redirection message

| HTTP code | Meaning              | 
|-----------|----------------------|
| 400       | Bad Request          | 

#### Client error code

| HTTP code | Meaning            | 
|-----------|--------------------|
| 300       | Multiple Choices   | 
| 301       | Moved permanently  |
| 302       | Found              |
| 307       | Temporary ReDirect |
| 308       | Permanent ReDirect |


#### Server error response

| HTTP code | Meaning               | 
|-----------|-----------------------|
| 501       | Internal Server error | 
| 502       | Bad Gateway           |
| 502       | Service Unavailable   |
| 504       | Gateway timeout       |
